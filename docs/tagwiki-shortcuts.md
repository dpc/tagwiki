# #Tagwiki #shortcuts

Tagwiki can be used mouse-free.

Available shortcuts:

* `n` for new page
* `e` for editing current page
* `d` for delete
* `alt+/` to jump to the search box, enter to 
* `ESC` for "going up" / "cancel"
* `j`, `k`, `h`, `l` to navigate links
* `alt+enter` or `ctrl+enter` to save
